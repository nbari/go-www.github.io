What it is www?

A web server that serves only static content, it can send traffic in plain HTTP, or HTTPS by using a self signed certificate localhost or using a valid certificate by using letsencrypt.

Why ?

Because of the need to share, test via HTTP/HTTPS the contents of a directory.

The idea came from the need to simplify this commands:

python -m SimpleHTTPServer

or

twistd -n web --path /foo/bar/baz
