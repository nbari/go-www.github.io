pre-compiled (Linux)

Using https://packagecloud.io/nbari/www

For deb:

curl -s https://packagecloud.io/install/repositories/nbari/www/script.deb.sh | sudo bash

Then:

apt install www

For rpm:

curl -s https://packagecloud.io/install/repositories/nbari/www/script.rpm.sh | sudo bash

Then:

yum install www

Download binary

linux amd64: https://dl.bintray.com/nbari/www/www_2.0.0_linux_amd64.tar.gz

See more: https://bintray.com/nbari/www/www/2.0.0
